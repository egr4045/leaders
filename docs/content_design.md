## 2. Общие кризисы (Авто-состояния)
- `golod` (Голод) — еда < 0
- `krizis` (Дефолт) — деньги < 0
- `protesty` (Массовые протесты) — недовольство населения

## 3. Глобальные Мем-ивенты (Глобальные статусы)
- `pandemic` (Пандемия / Ковид-26) — экономика вниз, наука вверх.
- `labubu_trend` (Мода на Лабубу) — экономика и довольство вверх, наука в ноль.
- `crypto_crash` (Крах Хомяка) — финансы обваливаются, умники плачут.
